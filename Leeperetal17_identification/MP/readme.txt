List of files in the folder \Leeperetal17_identification\MP


This folder mainly contains arbitrary precision version of files in \Leeperetal17_identification\Double:

comp_spec2_mp.m - computes spectral density of the medium scale monetary-fiscal model of Leeper et al (2017) at specified frequencies and parameter values in arbitrary precision.

kloptlpr2_mp.m - the objective function for global optimization search in arbitrary precision.

pfhlpr.m - computes empirical distance between two models in arbitrary precision.

transfmplpr.m - transformation to and from the unconstrained parameter form for fminsearch optimization in arbitrary precision.
