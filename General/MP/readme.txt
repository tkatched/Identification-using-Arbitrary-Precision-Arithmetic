List of files in the General\MP folder:

Matlab functions:


gensys_mp2.m - arbitrary precision implementation of the gensys algorithm modified to allow for indeterminacy. The modification follows Lubik and Schorfheide (2003). 

quadcomp.m - input type-dependent code to compute Gaussian quadrature in arbitrary or double precision.

svdrr_mp.m - arbitrary precision implementation of the Matlab translation of the Lubik and Schorfheide (2004) Gauss routine svdrr
