List of files in the Model files\SGU 2012\ MP folder:

Matlab functions:


sgusolv_mp.m - solves the baseline specification of the linearized model using gensys_mp2.m, with the original observables (log-differences)

sgusolvl_mp.m - solves the baseline specification of the linearized model using gensys_mp2.m, with the observables in levels

sgusolvl3_mp.m - solves the baseline and an MA shocks specification of the linearized model using gensys_mp2.m, with the observables in levels


Matlab scripts:

sgu_theta0mp.m - sets parameter values from SGU(2012) Table II posterior median in arbitrary precision

sgusspar_mp.m - auxiliary file for the arbitrary precision solution specifying steady state parameters

sgusetsysmat_mp.m - auxiliary file for sgusolv_mp.m setting up matrices for gensys.

sgusetsysmatl_mp.m - auxiliary file for sgusolvl_mp.m setting up matrices for gensys.

sgusetsysmatl3_mp.m - auxiliary file for sgusolvl3_mp.m setting up matrices for gensys.

Data files:

theta0mp_sgu.mat - benchmark model parameters generated by sgu_theta0mp.m.
